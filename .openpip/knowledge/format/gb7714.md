# GB/T 7714-2015 参考文献格式

## 类型标识
[J]期刊  [M]专著  [C]会议  [D]学位论文  [P]专利  [S]标准  [EB/OL]在线

## 格式模板

### 期刊[J]
作者. 题名[J]. 刊名, 年, 卷(期): 页码.

### 专著[M]
作者. 书名[M]. 出版地: 出版者, 年.

### 会议[C]
作者. 题名[C]//会议名. 出版地: 出版者, 年: 页码.

### 学位论文[D]
作者. 题名[D]. 城市: 学校, 年.

### 专利[P]
发明者. 题名: 国别, 专利号[P]. 年.

### 标准[S]
标准号 名称[S]. 年.

### 在线[EB/OL]
作者. 题名[EB/OL]. (日期)[引用日期]. URL.

### 报纸[N]
作者. 题名[N]. 报纸名, 年-月-日(版次).

### 数据库[DB/OL]
作者. 题名[DB/OL]. (日期)[引用日期]. URL.

## 作者规则
- 中文：全部列出，>3人用"等"
- 英文：姓在前名缩写，>3人用"et al."

## 标点
- 中文文献用中文标点
- 英文文献用英文标点

## 正文引用
- [1]  [1,2]  [1-3]  [1, p.45]

## 常见错误
| 错误 | 修正 |
|------|------|
| 张三李四. 论文 | 张三, 李四. 论文 |
| 缺类型标识 | 补充[J][M]等 |
| 标点混用 | 中文用中文标点 |
| 英文作者名全拼 | WANG X Y |
| 缺页码 | 补充起止页码 |
