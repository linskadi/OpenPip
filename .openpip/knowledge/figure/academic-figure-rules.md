# 学术图表规范

## 核心原则

1. **清晰优先**：信息传达 > 视觉美观
2. **可复现**：所有图表通过代码生成，不使用手绘或截图
3. **黑白兼容**：颜色仅作辅助区分，黑白打印仍可辨识

## 禁止事项

### 禁止的视觉效果
- ❌ 3D 效果（3D 柱状图、3D 饼图等）
- ❌ 阴影 / 投影
- ❌ 渐变填充
- ❌ 发光 / 发光效果
- ❌ 浮雕 / 立体边框
- ❌ 装饰性背景纹理

### 禁止的配色
- ❌ 高饱和荧光色（#FF00FF, #00FF00）
- ❌ 花哨渐变色
- ❌ 红-绿对比（色盲不友好）
- ❌ 纯彩虹色板（jet, rainbow）
- ❌ 超过 8 种颜色

## 分辨率要求

| 用途 | 最低 DPI | 推荐 DPI |
|------|---------|---------|
| 期刊投稿（矢量图） | N/A（PDF/SVG） | N/A |
| 期刊投稿（位图） | 300 | 600 |
| 学位论文 | 300 | 300 |
| 幻灯片 | 150 | 300 |

**代码示例**
```python
# PDF 矢量图（投稿用）
plt.savefig('figure.pdf', dpi=300, bbox_inches='tight')

# PNG 位图（预览用）
plt.savefig('figure.png', dpi=600, bbox_inches='tight')
```

### 色板选择原则

```python
# 推荐：使用固定色板
PALETTE = ['#3C5488', '#E64B35', '#00A087', '#4DBBD5', '#F39B7F']

# 禁止：使用自动配色
# plt.cm.Set3(...)  # 不可控
# plt.cm.rainbow(...)  # 色盲不友好
```

### 线条与标记区分

同一图表内多条线/柱，通过以下方式区分（优先级从高到低）：

1. **线条样式**：实线 `-`、虚线 `--`、点划线 `-.`、点线 `:`
2. **标记形状**：圆 `o`、方 `s`、三角 `^`、菱形 `D`、倒三角 `v`
3. **颜色**：最后才用颜色区分

```python
# 多系列区分示例
styles = [
    {'color': '#3C5488', 'marker': 'o', 'ls': '-'},
    {'color': '#E64B35', 'marker': 's', 'ls': '--'},
    {'color': '#00A087', 'marker': '^', 'ls': '-.'},
]
```

## 坐标轴规范

### 必须包含
- ✅ 坐标轴标签（含变量名和单位）
- ✅ 刻度值
- ✅ 刻度线（朝外）

### 推荐设置
- ✅ 隐藏上方和右侧边框 (`spines`)
- ✅ 适当的网格线（浅色虚线，可选）
- ✅ 零点标注（如适用）

### 常见错误

```python
# ❌ 错误：缺少单位
ax.set_xlabel('Temperature')

# ✅ 正确：包含单位
ax.set_xlabel('Temperature (°C)')

# ❌ 错误：标签无意义
ax.set_xlabel('X')

# ✅ 正确：标签描述清晰
ax.set_xlabel('Processing time (s)')
```

## 图例规范

### 位置优先级
1. 图表外部右上角（首选）
2. 图表内部空白区域
3. 图表下方

### 要求
- ✅ 框架可选（推荐关闭 `frameon=False`）
- ✅ 字号 ≤ 坐标轴标签
- ✅ 名称简洁（≤ 4 个汉字 / 8 个英文字符）
- ❌ 避免遮挡数据点

```python
# 推荐：无边框图例
ax.legend(frameon=False, fontsize=7, loc='upper left')
```

## 图注格式

### IEEE 风格
```
Fig. 1. 简短描述。详细说明，包含实验条件和参数。
```

### Nature 风格
```
**Figure 1** | 简短描述。详细说明。副标题信息。
```

### 中文期刊风格
```
图 1 简短描述。详细说明。
```

### 学位论文风格
```
图 X.Y 简短描述。详细说明，包含实验条件和参数信息。
```

## 常见错误示例

| 错误 | 问题 | 修正 |
|------|------|------|
| 3D 柱状图 | 虚假深度，误导读者 | 改为 2D 柱状图 |
| 彩虹色板 | 色盲不友好 | 使用学术安全色板 |
| 花哨渐变 | 分散注意力 | 使用纯色填充 |
| 隐藏零点的 Y 轴 | 夸大差异 | 从 0 开始或标注截断 |
| 超过 8 种颜色 | 难以区分 | 分面展示 |
| 无单位的坐标轴 | 信息不完整 | 添加单位 |
| 图例遮挡数据 | 可读性差 | 移至空白区域 |
| 彩色线条无标记 | 打印后无法区分 | 添加标记形状 |
| 过度拥挤的刻度 | 难以阅读 | 减少刻度数量 |
