# 灵敏度分析方法

## 概述

灵敏度分析（Sensitivity Analysis）是数学建模竞赛的核心要求之一。它检验模型输出对输入参数变化的敏感程度，是验证模型鲁棒性的重要手段。

## 为什么需要灵敏度分析

1. **验证模型可靠性**：参数估计总有误差，模型不应因微小变化而剧变
2. **识别关键参数**：找出对结果影响最大的参数，指导实际决策
3. **增强说服力**：展示模型在合理范围内都有效
4. **竞赛评分标准**：多数竞赛将灵敏度分析列为评分要点

## 方法一：单参数灵敏度分析

### 原理
固定其他参数不变，逐一改变目标参数，观察输出变化。

### 操作步骤
1. 确定待分析参数 $p_i$ 及其基准值 $p_i^0$
2. 设定变化范围（如 $\pm 10\%$, $\pm 20\%$, $\pm 50\%$）
3. 固定其他参数，仅改变 $p_i$
4. 记录每次的输出 $y_i$
5. 计算灵敏度系数

### 灵敏度系数计算

**绝对灵敏度**：
$$S_i = \frac{\partial y}{\partial p_i} \approx \frac{\Delta y}{\Delta p_i}$$

**相对灵敏度（弹性系数）**：
$$E_i = \frac{\partial y}{\partial p_i} \cdot \frac{p_i^0}{y^0} \approx \frac{\Delta y / y^0}{\Delta p_i / p_i^0}$$

**灵敏度系数含义**：
- $|E_i| > 1$：输出对该参数敏感（关键参数）
- $|E_i| = 1$：输出与参数等比例变化
- $|E_i| < 1$：输出对该参数不敏感

### 报告模板

```markdown
| 参数 | 基准值 | 变化范围 | 输出变化 | 灵敏度系数 | 敏感性判断 |
|------|--------|----------|----------|-----------|-----------|
| $p_1$ | 100 | ±10% | ±5% | 0.5 | 不敏感 |
| $p_2$ | 0.5 | ±10% | ±15% | 1.5 | 敏感 |
```

## 方法二：多参数灵敏度分析

### 原理
同时改变多个参数，观察输出的联合变化。

### 操作步骤
1. 选择 $k$ 个关键参数
2. 设定每个参数的变化范围
3. 生成参数组合（全因素或部分因素设计）
4. 对每个组合运行模型
5. 分析输出的分布

### 正交实验设计

当参数较多时，使用正交表减少实验次数：

```markdown
| 实验号 | $p_1$ | $p_2$ | $p_3$ | 输出 $y$ |
|--------|-------|-------|-------|----------|
| 1 | -1 | -1 | -1 | 10.2 |
| 2 | -1 | 0 | +1 | 12.5 |
| 3 | 0 | -1 | +1 | 11.8 |
| ... | ... | ... | ... | ... |
```

### 极差分析

计算每个参数的极差 $R_i$：
$$R_i = \max(y_i) - \min(y_i)$$

极差越大，该参数影响越大。

## 方法三：蒙特卡洛模拟

### 原理
对参数进行随机抽样，统计输出的分布特征。

### 操作步骤
1. 为每个参数指定概率分布（正态、均匀、三角等）
2. 从分布中随机抽样 $N$ 次（$N \geq 1000$）
3. 对每次抽样运行模型
4. 统计输出的均值、标准差、置信区间

### 分布选择

| 参数特征 | 推荐分布 | 说明 |
|---------|---------|------|
| 已知均值和方差 | 正态分布 $N(\mu, \sigma^2)$ | 最常见 |
| 仅知范围 | 均匀分布 $U(a, b)$ | 保守估计 |
| 有最可能值 | 三角分布 $Tri(a, m, b)$ | 工程常用 |
| 偏态分布 | 对数正态分布 $LN(\mu, \sigma^2)$ | 正值参数 |

### 输出分析

```markdown
## 蒙特卡洛模拟结果

| 统计量 | 值 | 说明 |
|--------|-----|------|
| 样本数 | 10000 | 抽样次数 |
| 均值 | 125.3 | 期望输出 |
| 标准差 | 12.7 | 输出波动 |
| 95%置信区间 | [100.4, 150.2] | 95%的输出落在此范围 |
| 变异系数 | 10.1% | 相对波动程度 |
```

## 方法四：龙卷风图

### 原理
用条形图直观展示各参数对输出的影响程度，按影响大小排序。

### 绘制方法
1. 对每个参数，计算输出在参数变化范围内的最大值和最小值
2. 以基准输出为中心，画出每个参数的影响范围
3. 按影响范围从大到小排列

### 代码示例（Python）

```python
import matplotlib.pyplot as plt
import numpy as np

params = ['参数A', '参数B', '参数C', '参数D']
mins = [90, 95, 98, 99]
maxs = [130, 120, 110, 105]
baseline = 110

fig, ax = plt.subplots(figsize=(10, 6))
y_pos = np.arange(len(params))

for i in range(len(params)):
    ax.barh(y_pos[i], maxs[i] - baseline, left=baseline, height=0.5, color='red', alpha=0.7)
    ax.barh(y_pos[i], baseline - mins[i], left=mins[i], height=0.5, color='blue', alpha=0.7)

ax.set_yticks(y_pos)
ax.set_yticklabels(params)
ax.set_xlabel('输出值')
ax.set_title('龙卷风图（Tornado Diagram）')
ax.axvline(x=baseline, color='black', linestyle='--', label='基准值')
ax.legend()
plt.tight_layout()
plt.savefig('tornado.png', dpi=300)
```

## 方法五：局部灵敏度分析（解析法）

### 原理
对模型方程直接求偏导数，得到解析形式的灵敏度。

### 适用条件
- 模型方程可解析表达
- 偏导数可计算
- 参数变化范围较小

### 示例

若模型为 $y = f(x_1, x_2, ..., x_n)$，则：

$$S_i = \frac{\partial f}{\partial x_i}$$

对于无法直接求导的模型，可用数值近似：

$$S_i \approx \frac{f(x_i + \Delta x_i) - f(x_i - \Delta x_i)}{2\Delta x_i}$$

## 常见错误

1. **❌ 只分析单参数**：应同时考虑单参数和多参数分析
2. **❌ 变化范围不合理**：应基于实际数据确定合理范围
3. **❌ 不给出灵敏度系数**：必须量化灵敏度，不能只说"敏感"或"不敏感"
4. **❌ 缺少结论**：必须从灵敏度分析得出实际建议
5. **❌ 蒙特卡洛抽样不足**：至少 1000 次，推荐 10000 次
6. **❌ 未讨论参数交互**：多参数分析应考虑参数间的相互影响

## 竞赛灵敏度分析规范

- 灵敏度分析不是可选项，是竞赛论文的必要内容。
- 方法：单参数扰动（±10%、±20%）观察目标函数变化；或 Tornado 图展示参数影响力排序。
- 对优化类问题：改变约束右端项或目标系数，观察最优解是否发生质变。
- 对预测类问题：改变关键假设或初始条件，观察预测曲线漂移幅度。
- 结论：若小扰动导致结果大幅变化，说明模型对该参数敏感，需在论文中声明。
