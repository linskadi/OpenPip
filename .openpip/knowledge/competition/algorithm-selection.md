# 算法选择指南

## 方法选择决策树

| 问题类型 | 推荐方法 | 避免方法 | 原因 |
|---------|---------|---------|------|
| 多目标优化 | NSGA-II / MOEA/D | 枚举法 / 加权求和 | 枚举法在变量>5时不可行 |
| 分类/预测（小样本n<500） | GLM / 有序Logit / SVM | 随机森林 / XGBoost | 小样本下树模型易过拟合 |
| 回归（小样本） | 岭回归 / Lasso / 弹性网络 | 普通最小二乘 | 多重共线性导致不稳定 |
| 调度/路径规划 | ALNS / 遗传算法 / 模拟退火 | 穷举 / 贪心 | 穷举指数爆炸 |

## 小样本设置的特殊处理（n < 500）

1. **必须使用正则化**：L1(Lasso) / L2(Ridge) / ElasticNet
2. **必须交叉验证**：至少 5-fold CV，报告均值±标准差
3. **必须报告置信区间**：95% CI
4. **避免复杂模型**：随机森林/GBDT 需显式设置 `min_samples_leaf`, `max_depth`
5. **特征选择优先**：先降维再建模

## 多目标优化规范

- 变量数 > 5 时禁止枚举法
- 必须使用 Pareto 前沿可视化
- 报告 HV (Hypervolume) 指标
- 提供 3-5 个代表性解供决策者选择

## 模型验证要求

### 统计检验规范
- 所有对比实验必须报告 mean±std
- 必须提供 p-value 或 95% 置信区间
- 灵敏度分析至少覆盖 3 个关键参数
- 扰动范围：0.8x ~ 1.2x，5 个水平

### 模型选择论证
- 每个模型选择必须说明"为什么选A而非B"
- 对比至少 2 种备选方法
- 从性能、复杂度、可解释性三个维度论证

## 算法实现要求

### 必须实现的算法类型（至少选一种）

1. **自定义优化算法**：
   - ALNS（自适应大邻域搜索）：破坏算子→修复算子→局部搜索→模拟退火接受
   - 遗传算法：编码→交叉→变异→选择→适应度评估
   - 模拟退火：邻域生成→接受准则→降温调度

2. **灵敏度分析**（必须包含）：
   - 至少3个关键参数的单因素扰动（0.8x ~ 1.2x，5个水平）
   - 龙卷风图（tornado chart）：参数敏感性排序
   - 鲁棒性带图（robustness band）：扰动下的成本波动范围

3. **模型检验**（必须包含）：
   - 约束回代验证（检查所有约束是否满足）
   - 收敛曲线（目标函数 vs 迭代次数，含当前解+历史最优）
   - 可行性指标（任务覆盖率、容量违约数、约束满足率）
