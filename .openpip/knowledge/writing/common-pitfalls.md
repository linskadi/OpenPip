# 常见学术写作陷阱（Common Pitfalls）

> 由 OpenPip 审稿反馈系统自动维护。每次审稿发现的共性问题经用户确认后沉淀于此。

## 贡献声明相关

- **贡献模糊**：核心贡献无法用一句话说清楚 → 使用 contribution-architect 精炼
- **Over-claiming**：使用 "first"/"novel"/"SOTA" 但缺充分比较 → 必须有实验/引用支撑
- **Scope mismatch**：声称解决宽泛问题但实验仅在单一场景验证 → 缩小 scope 或增加实验

## 论证相关

- **证据缺失**：每个 claim 必须有对应实验数据或引用 → 逐段检查 Claim→Evidence 对齐
- **循环论证**：方法解决了 X，但 X 定义依赖于该方法 → 重新审视定义独立性
- **因果混淆**：correlation → causation 缺乏控制变量 → 补充消融实验或讨论替代解释

## 实验相关

- **基线不全**：只对比 SOTA 无 naive baseline → 至少包含 naive + SOTA + ablation 三类基线
- **无消融**：多个组件的方法无消融实验 → 每个组件至少一个消融
- **无统计显著性**：报告均值无比标准差/置信区间 → 分类用 McNemar, 回归用 paired-t
- **选择性报告**：只报告最好结果不报所有实验 → 完整报告全部实验，失败也是一种贡献

## 写作相关

- **被动语态混用**：被动/主动混用 → 通篇统一（推荐被动语态）
- **术语不一致**：同一概念出现中/英文/缩写混用 → 首次定义后统一使用
- **引用不足**：段落无引用或少引用 → 每段 2-4 篇引用，按 GB/T 7714
- **章节脱节**：章节间无过渡句 → 每章末总结+预告，每章首回顾+引出
