# 议论文论证结构规范

## 概述

议论文（Argumentative Essay）的核心是提出明确的论点，用充分的证据和严密的逻辑进行论证，最终说服读者。适用于各类议论文竞赛（如 John Locke、NYT 写作竞赛）以及英文学术写作。

## 论证框架

### 经典五段式结构
```
第 1 段：引言
  └── 背景句 → 问题陈述 → 论点句（Thesis Statement）

第 2 段：论据 1
  └── 主题句 → 证据 → 解释 → 过渡

第 3 段：论据 2
  └── 主题句 → 证据 → 解释 → 过渡

第 4 段：反驳与让步
  └── 承认反方观点 → 反驳 → 重申本方立场

第 5 段：结论
  └── 重申论点 → 总结论据 → 升华/呼吁
```

### 论点的 3C 质量标准
- Clear：一个段落只有一条核心论点
- Contestable：论点有讨论空间，不是事实陈述
- Central：每条论点都支持中心论点

## 论证方法

### 逻辑论证（Logos）
- 演绎法：大前提→小前提→结论
- 归纳法：具体案例→一般规律
- 类比法：相似案例推理

### 证据类型（按说服力排序）
1. 统计数据：权威机构的一手数据
2. 学术文献：同行评审的研究成果
3. 案例分析：具体事例的深度分析
4. 专家证词：领域权威的观点
5. 逻辑推理：基于事实的严密推导

### 修辞策略
- 反问：引发读者思考
- 排比：增强节奏感
- 对比：突显差异

## 反驳技巧

### 四步反驳法
1. 承认：公平表述反方观点
2. 限制：指出反方观点的适用边界
3. 反驳：给出反方观点的逻辑漏洞
4. 回归：回到本方立场

### 反驳类型
- 事实反驳：反方依据的事实不成立
- 逻辑反驳：反方的推理有误
- 价值反驳：反方的价值优先级有问题
- 后果反驳：反方方案会导致不良后果

## 常见逻辑谬误

- 稻草人：曲解对方观点再攻击
- 滑坡谬误：假设一步走错就会万劫不复
- 诉诸情感：用情绪代替论证
- 虚假二分：只给两个极端选项
- 循环论证：论点本身就是论据
- 人身攻击：攻击人而非观点

## 写作要点

- 每段只论证一个观点
- 证据后必须有解释（Evidence+Explanation）
- 避免情绪化语言，保持理性客观
- 承认反方观点可以增强说服力
- 结论不要引入新论据
