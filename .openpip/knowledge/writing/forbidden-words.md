## 学术写作常见错误
| 错误类型 | 示例 | 修正 |
|----------|------|------|
| 自我指代过多 | "本文/我们"过多 | 适当使用被动句 |
| 时态混乱 | 方法部分用过去时，结论用现在时 | 保持一致 |
| 冠词缺失 | "system is designed" | "the system is designed" |
| 缩写未定义 | 首次出现PID | 首次出现需全称+缩写 |
| 图表引用不当 | "如图所示"（未指明图号） | "如图1所示" |
| 数据精度不一 | "3.14"和"3.1416"混用 | 同文精度统一 |
| 单位格式错误 | "5mm" | "5 mm" |
| 中英文混排 | 中文间插入英文标点 | 统一标点 |
| 重复表述 | 同一意思换词重复 | 删除冗余 |
| 指代不清 | "该方法"前文有多种方法 | 明确指代对象 |
