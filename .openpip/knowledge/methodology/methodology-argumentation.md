# 论证逻辑方法

## 论证单元
论点(Claim) → 论据(Evidence) → 论证(Reasoning) → 结论(Conclusion)

## 论证方法

### 演绎（一般→特殊）
大前提 → 小前提 → 结论

### 归纳（特殊→一般）
观察1 + 观察2 + 观察3 → 一般结论

### 类比（相似→推断）
现象A有效 → 相似性 → 推断B也可能有效

### 因果（原因→结果）
原因 → 机制 → 结果

## 常见逻辑错误
| 错误 | 示例 | 修正 |
|------|------|------|
| 循环论证 | "因为A有效所以A有效" | 提供独立证据 |
| 以偏概全 | "数据集X有效→所有场景有效" | 限定范围 |
| 诉诸权威 | "专家说有效" | 提供实验证据 |
| 假因果 | "A之后B→A导致B" | 排除其他因素 |

## 章节间衔接
- 绪论→理论："为解决上述问题，本章介绍相关理论基础"
- 理论→方法："基于上述理论，本文提出以下方法"
- 方法→实验："为验证方法有效性，设计以下实验"
- 实验→结论："基于实验结果，本文得出以下结论"

---

## 多步论证链构建

### 论证链模板
```
前提1（已知/引用） + 前提2（实验/观察） → 推理步骤 → 结论
```

### 因果论证 vs 相关性论证
- **因果论证**："X 导致 Y"——需要控制变量实验或因果推断方法
- **相关性论证**："X 与 Y 相关"——只需要统计显著性
- ❌ 常见错误：把相关性当因果性

### 如何处理反面证据
1. 承认反面证据存在
2. 分析为什么在特定条件下出现
3. 说明本文方法如何缓解或规避
4. 标记为 Limitations（如果是真实局限）
